Lego Fanatics

A project plan for a Lego fan page.

Site Name

This name represents a product fan base where fans can learn and connect.

Site Purpose

The purpose of this site is for fans to connect, learn, and find new ways to have fun with Lego, such as Lego events, theme parks, movies, and the TV show Lego Masters.

Scenarios

• Where can I find retired Lego sets?

• Where and how did Lego get started?

• Where can I meet other Lego fans?

• Where can I find missing Lego pieces?
